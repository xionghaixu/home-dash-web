# home-dash-web

## 介绍

`home-dash-web` 是当前仓库中的前端项目目录名，对应原 `PND`（`Personal Network Disk`）项目的前端工程，当前基于 Vue 3 构建，负责文件工作台、上传交互、系统信息展示和视频播放等页面能力。

当前版本以“核心文件工作台”作为第一阶段主线，后续会继续演进到预览检索、媒体中心、数据治理和家庭设备联动界面。

## 当前定位

- 技术栈：Vue 3 + Vite + Element Plus + Pinia
- 角色定位：单体后端配套前端
- 适用场景：单用户、家庭主人、局域网优先
- 当前重点：功能完善、页面完善、交互闭环
- 第六阶段再统一处理登录、安全、域名和 SSL 相关页面

## 当前已实现页面与能力

- 文件列表页
- 全局上传面板
- 文件夹树选择弹窗
- 系统信息页
- 视频播放页
- 404 页面
- 文件上传、新建、重命名、删除、移动、复制、下载
- 文件列表排序与面包屑路径导航
- 文件类型图标映射、文件大小与时间格式化

## 技术栈

- Vue：3.5.13
- Vue Router：4.5.0
- Pinia：2.3.0
- Element Plus：2.9.0
- Vite：6.0.0
- Axios：1.7.7
- Node.js：>= 20.0.0

## 快速开始

```bash
npm install
npm run dev
npm run build
npm run preview
```

## 规划方向

- 阶段一：核心文件工作台补齐与体验打磨
- 阶段二：预览与检索整理
- 阶段三：媒体中心与家庭 NAS 体验
- 阶段四：数据治理与任务中心
- 阶段五：家庭设备与备份页面
- 阶段六：登录与安全页面
